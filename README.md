## 💻 Development Setup

To get started with development, follow these steps:

1. **Clone the repository**:
    ```sh
    git clone https://github.com/gprs1022/API-call-in-flutter.git
    cd API-call-in-flutter
    ```

2. **Install dependencies**:
    ```sh
    flutter pub get
    ```

3. **Run the application**:
    ```sh
    flutter run
    ```
